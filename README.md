# Element Plus Snippets

基于 `Element Plus` 实现的一款代码提示插件

语法采用 `4` 个空格缩进的形式

## 特性

-   开箱即用：安装插件后，即可直接在代码中使用

-   组件支持：提供了大部分常用组件的语法提示

## 开始第一个语法提示

在 `.vue` 文件中输入 `el-` 时就会出现一系列 `Element Plus` 组件的代码提示。
